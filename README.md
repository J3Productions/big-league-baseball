# Big Leauge Baseball

Port of the 3M Sports board game written in Javascript using Express.

### Install Dependencies

`npm install`

### Run the Server

`npm run test`

To view the server, view `localhost:3000` in your favorite browser.

### Generate Documentation

`npm run doc`
